# 01. 프로젝트 준비

### 개발 환경
    Linux mint cinnamon 22.2 "Zara"
    Visual Studio Code 1.106
    Jupyter v2025.9.1
    Python 3.12.3

### Step 1. 프로젝트 폴더 생성.
```bash
mkdir -p ./CardSpendingData
cd CardSpendingData
```

### Step 2. 가상환경 생성.
```bash
# 1. 프로젝트 하위 폴더 생성.
mkdir -p ./{data/{raw,processed,external},outputs/{charts,images,tables}}

# 2. 가상환경 생성.
python3 -m venv venv

# 3. 가상환경 활성화.
source venv/bin/activate
```

### Step 3. 패키지 설치
```bash
# 1. pip 업그레이드
pip install --upgrade pip

# 2. 필수 패키지 설치
pip install pandas numpy scipy matplotlib seaborn scikit-learn

# 3-1. Jupyter notebook 관련 패키지
## Visual Studio Code의 "Jupyter Extension Pack" 설치
pip install jupyter jupyterlab ipython

# 3-2. 한글 폰트 설정
pip install fonts-nanum
```
